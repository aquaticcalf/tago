# js api

## functions

### log
generates console.log for the given string

```go
import "aqclf.xyz/tago/js"

log := js.log("hello world")
// outputs: console.log("hello world")
```

### alert
generates alert for the given string

```go
import "aqclf.xyz/tago/js"

alert := js.alert("message")
// outputs: alert("message")
```
